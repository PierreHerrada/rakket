Getting Started

Prerequisites

  • Docker and Docker Compose installed
  • A Slack workspace where you have permission to install apps

Quick Start

# 1. Clone the repo
git clone https://github.com/YOUR_USER/rakket.git
cd rakket

# 2. Configure environment
cp .env.example .env
# Edit .env with your Slack credentials

# 3. Start everything
docker-compose up -d

The app will be available at http://localhost:8080.

First Steps

  1. Create a Slack app — Follow the Slack Setup Guide.
  2. Log in — Visit the web UI and authenticate with Slack OAuth.
  3. Make yourself admin — Set is_admin = true for your player in the database.
  4. Configure tournament schedule — Set the day and time in your .env file.
  5. Wait for tournament day — The bot will automatically post a registration message.

Development Setup

# Start only the database
docker-compose up postgres -d

# Backend (requires JDK 21)
cd backend
./gradlew run

# Frontend (requires Node.js 20+)
cd frontend
npm install
npm run dev

The frontend dev server runs on http://localhost:5173 and proxies API requests to the backend on port 8080.
