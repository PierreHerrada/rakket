Rakket

Your office ping pong league, automated. Self-hosted tournament manager with ELO rankings, Swiss brackets, achievements, and a Slack bot.

Get Started

Swiss-System Tournaments

Fair matchups using the Dutch pairing variant. Automatic bracket generation with rematch avoidance and bye handling.

ELO Rankings

Track skill progression with a proper rating system. K-factor adapts based on player experience level.

Slack Integration

Registration, score reporting, standings, and notifications right in Slack. Socket Mode for easy self-hosting.

Achievements

16+ unlockable badges for milestones: Champion, On Fire, Giant Killer, Comeback Kid, and more.

Player Profiles

Stats, ELO charts, head-to-head records, win streaks, and match history for every player.

Single Docker Image

Ships as one container. Multi-stage build with Kotlin backend, React frontend, and PostgreSQL.
