Slack Setup Guide

1. Create a Slack App

  1. Go to api.slack.com/apps
  2. Click "Create New App" → "From scratch"
  3. Name it "Rakket" and select your workspace

2. Enable Socket Mode

  1. Go to Socket Mode in the sidebar
  2. Enable Socket Mode
  3. Give the app-level token a name (e.g., "rakket-socket") and save it

3. Configure Bot Scopes

  1. Go to OAuth & Permissions
  2. Under Bot Token Scopes, add:
    • chat:write — Post messages
    • reactions:read — Read reactions for tournament registration
    • commands — Slash commands
    • users:read — Read user profiles
  3. Under User Token Scopes, add:
    • openid — OAuth login
    • profile — User profile info

4. Create Slash Command

  1. Go to Slash Commands
  2. Click "Create New Command"
  3. Command: /rakket
  4. Request URL: https://your-domain.com/api/slack/commands
  5. Description: "Ping pong tournament commands"
  6. Usage hint: "score @opponent 11-7 9-11 11-5"

5. Configure OAuth Redirect

  1. Go to OAuth & Permissions
  2. Add redirect URL: https://your-domain.com/api/auth/callback

6. Install the App

  1. Go to Install App
  2. Click "Install to Workspace"
  3. Copy the Bot User OAuth Token (starts with xoxb-)

7. Configure Environment Variables

Copy these values to your .env file:

  • SLACK_BOT_TOKEN — Bot User OAuth Token
  • SLACK_SIGNING_SECRET — From Basic Information → App Credentials
  • SLACK_CLIENT_ID — From Basic Information → App Credentials
  • SLACK_CLIENT_SECRET — From Basic Information → App Credentials
  • SLACK_CHANNEL_ID — Right-click the channel → "Copy link" → extract the ID

8. Invite the Bot

In your Slack channel, type /invite @Rakket to add the bot.
