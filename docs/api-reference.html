API Reference

All endpoints are prefixed with /api. Authentication is via Slack OAuth session cookie.

Health

MethodPathDescription
GET/api/healthHealth check

Authentication

MethodPathDescription
GET/api/auth/slackStart Slack OAuth flow
GET/api/auth/callbackOAuth callback
GET/api/auth/meCurrent user info
POST/api/auth/logoutLogout

Tournaments

MethodPathDescription
GET/api/tournamentsList all tournaments (paginated)
GET/api/tournaments/currentCurrent active tournament
GET/api/tournaments/:idTournament detail with rounds & matches
POST/api/tournaments/:id/scoreReport a match score

Leaderboard

MethodPathDescription
GET/api/leaderboard?type=elo|points&period=4w|12w|allRanked leaderboard

Players

MethodPathDescription
GET/api/playersList all players
GET/api/players/:idPlayer profile
GET/api/players/:id/statsFull player stats
GET/api/players/:id/matchesMatch history (paginated)
GET/api/players/:id/h2hHead-to-head records
GET/api/players/:id/badgesEarned badges
GET/api/players/:id/elo-historyELO rating over time

Admin

Requires is_admin = true on the player record.

MethodPathDescription
POST/api/admin/tournament/createManually create a tournament
PUT/api/admin/match/:id/scoreOverride a match score
PUT/api/admin/match/:id/disputeResolve or reset a disputed match
GET/api/admin/settingsView current app settings

Slack

MethodPathDescription
POST/api/slack/commandsSlack slash command handler
