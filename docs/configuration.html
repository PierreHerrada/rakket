Configuration

All configuration is done via environment variables. Copy .env.example to .env and edit the values.

Database

VariableDefaultDescription
DATABASE_URLjdbc:postgresql://postgres:5432/rakketPostgreSQL JDBC connection URL
DATABASE_USERrakketDatabase username
DATABASE_PASSWORDchangemeDatabase password

Slack

VariableDefaultDescription
SLACK_BOT_TOKEN—Bot User OAuth Token (xoxb-...)
SLACK_SIGNING_SECRET—Signing secret from Slack app settings
SLACK_CLIENT_ID—OAuth client ID
SLACK_CLIENT_SECRET—OAuth client secret
SLACK_CHANNEL_ID—Channel ID where the bot posts (C...)
